Description
-----------
The **Elementor Image, Title & Subtitle Widget** is a customizable widget for Elementor that allows you to add and style an image, title, and subtitle on your website. With this widget, you can adjust typography settings, image sizes, and more, directly from the Elementor editor.

Key Features:
- Add an image, title, and subtitle with a single widget.
- Customize typography for title and subtitle, including font family, size, weight, and color.
- Control image size with various options.
- Fully responsive and compatible with the latest Elementor version.
- Easy-to-use interface integrated with Elementor.

Installation
------------
1. Upload the plugin files to the `/wp-content/plugins/elementor-image-title-subtitle` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the widget to any Elementor page by searching for "Image Title Subtitle" in the Elementor widget panel.

Usage
-----
1. Open a page or post with Elementor.
2. Find the "Image Title Subtitle" widget in the Elementor widget panel.
3. Drag the widget onto your page.
4. Use the widget settings to upload an image, enter a title and subtitle, and customize typography and image size.

Frequently Asked Questions
--------------------------
Q: How do I customize the typography for the title and subtitle?
A: The widget provides options for adjusting the font family, size, weight, and color for both the title and subtitle in the Elementor editor.

Q: Can I change the image size?
A: Yes, you can choose from standard image sizes or specify custom dimensions.

Q: Is the widget responsive?
A: The widget is designed to be responsive and will adapt to different screen sizes.
